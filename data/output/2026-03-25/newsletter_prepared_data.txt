3. ‘CanisterWorm’ Springs Wiper Attack Targeting Iran

A new cyber attack, known as the "CanisterWorm", has been launched by a group called TeamPCP, targeting systems in Iran by spreading through poorly secured cloud services and wiping data on infected systems. This attack is particularly concerning because it not only steals data but also destroys it, leaving victims with little to no chance of recovery. The group is financially motivated and has been active since December 2025, compromising corporate cloud environments and extorting victims. Anyone using cloud services, especially those with connections to Iran or using Farsi as their default language, is at risk. The attack highlights the importance of securing cloud services and being vigilant against cyber threats.

The CanisterWorm attack is a type of wiper campaign that spreads through poorly secured cloud services, targeting systems that use Iran's time zone or have Farsi set as the default language. This campaign is the latest move by TeamPCP, a relatively new cybercrime group that has been active since December 2025. The group has been compromising corporate cloud environments using a self-propagating worm that exploits vulnerabilities in Docker APIs, Kubernetes clusters, Redis servers, and the React2Shell vulnerability. This allows them to move laterally through victim networks, siphoning authentication credentials and extorting victims over Telegram.

The victims of the CanisterWorm attack are primarily cloud service users, particularly those with connections to Iran or using Farsi as their default language. The attack has significant implications for various sectors, including finance, healthcare, and government, where sensitive data is often stored in cloud services. According to experts, TeamPCP's attacks have predominantly targeted cloud infrastructure, with Azure and AWS accounting for 97% of compromised servers. TeamPCP's strength lies in its ability to industrialize existing vulnerabilities, misconfigurations, and recycled tooling into a cloud-native exploitation platform. The group's use of automation and integration of well-known attack techniques has made them a significant threat to cloud service users.
